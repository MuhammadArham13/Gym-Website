# The Best CrossFit Gym

A responsive single-page landing site for a NYC CrossFit gym — built with plain HTML, CSS, and JavaScript.

🔗 Live: https://gym-site-six-xi.vercel.app/
📦 Repo: https://github.com/MuhammadArham13/Gym-Website

## Features

- Sticky navbar with scroll-spy active-link highlighting
- Mobile hamburger menu
- Hero section with parallax background
- Scroll-reveal animations (IntersectionObserver)
- Benefits, About, Team, Testimonial, and Gallery sections
- Newsletter signup form (front-end only)
- Back-to-top button
- Fully responsive (mobile, tablet, desktop)
- Accessible: ARIA labels, reduced-motion support, semantic HTML

## Tech Stack

HTML5 · CSS3 (custom properties, no framework) · Vanilla JavaScript · Google Fonts (Montserrat, Open Sans)

## Project Structure

\`\`\`
index.html    Page markup and content
style.css     Design tokens, layout, responsive styles
script.js     Navbar, mobile menu, smooth scroll, animations, forms
images/       Site imagery (logo, hero, team, gallery)
\`\`\`

## Running Locally

Just open \`index.html\` in a browser — no build step required.

## Deployment

Deployed as a static site on Vercel via the Vercel CLI:

\`\`\`bash
vercel
vercel --prod
\`\`\`
